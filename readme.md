# 🐍 Snake Game (JavaScript)

A simple Snake Game built using **HTML, CSS, and JavaScript** while learning core JavaScript concepts.

---

## 🚀 About the Project

This project was created as part of my JavaScript learning journey.
I followed a tutorial to understand:

* Snake movement logic
* Board/grid layout
* Food generation logic

After that, I **implemented several features on my own without following any tutorial**, which helped me understand how to write logic independently.

---

## ✨ Features

* 🎮 Classic Snake Game
* ⚡ Multiple Difficulty Modes (Easy / Medium / Hard) *(self-implemented)*
* ⏯️ Play / Pause functionality *(self-implemented)*
* 🧮 Score tracking
* 🏆 High Score using localStorage
* ⏱️ Timer (MM:SS format)
* 🔁 Restart Game

---

## 🧠 What I Learned

* How to structure game logic in JavaScript
* Working with `setInterval` and game loops
* Handling user input (keyboard events)
* Managing game state (start, pause, game over)
* Using `localStorage` for persistent data
* Converting logic into real working features

---

## 📱 Responsiveness

The UI is simple and I focused more on logic than design.
The game works **okay on mobile devices**, though it is mainly optimized for desktop.
